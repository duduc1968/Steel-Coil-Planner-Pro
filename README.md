# Steel Coil Planner Pro v3.0 beta

This version starts the Pattern Builder workflow.

## New in v3.0 beta
- Row Arrangement can use preset, Custom / Manual, or Pattern Builder.
- Pattern Builder creates arrangements from structured fields: Bottom port, Bottom starboard, Wedge, Upper row.
- The builder produces a calculation-ready pattern such as `4+4 / Wedge / 5`.
- Cargo list uses English terminology: `ID`, `Width`, `Weight`, optional `Diameter`.
- Values are auto-converted to metres and tonnes.

## Example cargo columns
`ID, Width, Weight, Diameter`

Width and Diameter may be in mm or m. Weight may be in kg or t.
